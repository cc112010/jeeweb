JeeWeb Generator 是一款基于SpringBoot+Spring+Mybatis+Mybatis Plus的智能的、可视化的、多数据源的代码生成平台；

JeeWeb Generator的开发理念是能用模板自动生成的尽量用模板实现，可以快速协助java开发人员解决40%的重复工作，让开发人员更多关注业务逻辑的实现，解放JAVA开发人员的开发压力，提高开发效率，为企业节省项目研发成本，减少开发周期。
